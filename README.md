# 项目&作品
# 航空器维修无人车平台
截取帧图片上传至服务器，进行双目摄像头测距，以及卷积神经网络测试叶片损伤，并返回相关数据。
使用树莓派作为控制板，搭载双目摄像头，Flask作为服务器，后台代码使用Python，前台使用JS，卷积神经网络使用Pytorch。
航线维修增强现实系统
